mpm

A roughly npm install-compatible package manager for Node.JS in ~2001 lines of code (eLOC).

  • implements npm's basic install algorithm
  • correctly resolves (circular) dependencies
  • supports semver
  • correctly deals with devDependencies
  • interfaces with the npm registry
  • bootstrapping - That's why node_modules is checked in. It's fine, really.
  • fast - About twice as fast when installing express. Benchmarks are flawed. Try yourself and don't blame me.

Installation

  git clone https://github.com/alexanderGugel/mpm mpm && cd $_ && make install

Usage

  mpm - An alternative package manager for Node.js

  Usage:
    mpm install <pkg>
    mpm install <pkg>@<version>
    mpm install <pkg>@<version range>

    Can specify one or more: mpm install semver@^5.0.1 tape
    If no argument is supplied, installs dependencies from package.json.
    `mpm i` also works.

  Flags:
    -h, --help      Show usage information and exit

  README:  https://github.com/alexanderGugel/mpm
  ISSUES:  https://github.com/alexanderGugel/mpm/issues
